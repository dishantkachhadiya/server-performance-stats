# Server Performance Stats

A Bash script that analyzes basic server performance statistics.

## Features

- CPU Usage
- Memory Usage
- Disk Usage
- Top 5 CPU-consuming processes
- Top 5 Memory-consuming processes
- System Uptime
- OS Version

## Requirements

- Linux
- Bash

## Usage

```bash
chmod +x server-stats.sh
./server-stats.sh
```
